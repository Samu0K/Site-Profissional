# 🚀 Meu Portfólio Pessoal

Este é o meu projeto de portfólio desenvolvido durante o curso de **Análise e Desenvolvimento de Sistemas**. O foco principal foi criar uma interface moderna, responsiva e com funcionalidades de experiência do usuário (UX).

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Estrutura semântica)
* **CSS3** (Layouts com Flexbox e Variáveis para Temas)
* **JavaScript** (Lógica de Dark Mode e Manipulação do DOM)

## ✨ Funcionalidades
- [x] **Dark Mode:** Troca dinâmica de temas (Light/Dark) com persistência via `localStorage`.
- [x] **Switch Personalizado:** Ícones de Sol e Lua que acompanham a troca de tema.
- [x] **Layout Responsivo:** Adaptado para dispositivos móveis e desktop.
- [x] **Formulário de Contato:** Organizado com Flexbox e estilização moderna.

## 📱 Outros Conhecimentos
Além do desenvolvimento web, possuo conhecimentos em **Desenvolvimento Android**, focando em:
* Criação de layouts XML.
* Ciclo de vida de aplicações.
* Interface do usuário seguindo padrões Material Design.

## 📁 Como visualizar o projeto
1. Clone este repositório: `git clone https://github.com/Samu0K/Site-Profissional`
2. Abra o arquivo `index.html` no seu navegador.
